# clasp-container

VSCodeのRemote-DevelopmentでGASを書くためのひな型<br>
PCにはnodeを入れたくなかった

# Features

VSCode + Remote-Containers + Clasp で GASの管理


# Requirement

* VSCode
    * Remote-Development

# Usage

```bash
git clone https://github.com/ta-kondo/clasp-container.git
cd clasp-container
code .
```

Remote-Containersでコンテナを起動し、マウントされたフォルダーでGASのプロジェクトをClaspで作る<br>
解説は略
